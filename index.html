RUEquivalent

I developed a website designed to streamline the process for Rutgers students seeking course equivalencies across New Jersey's 18 community colleges. This platform significantly reduces the time students would otherwise spend navigating the NJ Transfer website, where they must manually search and scroll through numerous combinations of community colleges to identify the correct course equivalencies. To implement the website, I found the Rutgers courses API in json format. Then I used Selenium and beatiful soup to scrape the course equivalencies from the NJ Website. I used Open AI's GPT 3.5 model to generate course descriptions and the Google MAPS to get the distance from each community college to the user location. One of the challenges that I faced scraping course equivalencies from the NJ Transfer website. I was attempting to submit a post request to the website to get the course equivalencies, but I was getting the same response. Eventually, I used Selenium to click the button, which resolved the issue. This project has taught me to be resourceful and to be willing to try different approaches to solve a problem, even if it doesn't apear that the new approach will resolve the issue.

Fire Extinguishing Bot

In this project, I tested various bots navigating a maze to reach a button while dealing with a spreading fire, using different algorithms to handle the challenge. Bot 1 employs Breadth-First Search (BFS) to plan a static path to the button, avoiding only the initial fire cell and ignoring the spread of fire. Bot 2 also uses BFS but adapts dynamically by re-planning the shortest path at each step, avoiding current fire cells. Bot 3 further refines this approach by considering both current and adjacent fire cells when re-planning, and if no viable path is found, it resorts to planning based on the current fire cells. Bot 4 utilizes the A* algorithm with a Pythagorean heuristic to re-plan the path at each step, effectively navigating around current fire cells. This setup allows for a comparison of how each algorithm performs in adapting to a dynamic, changing environment.
